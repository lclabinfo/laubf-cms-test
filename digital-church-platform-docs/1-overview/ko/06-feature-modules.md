# 06. 핵심 기능 모듈 상세 명세

## 개요

각 솔루션 모듈의 상세 기능 명세서입니다. 모듈별 핵심 기능, 데이터 모델 개요, UI 구성 요소를 정의합니다.

---

## 1. Website Module (필수)

### 개요

교회 공식 웹사이트 구축 및 관리를 위한 핵심 모듈입니다. 테넌트별 커스터마이징이 가능한 템플릿 시스템을 기반으로 합니다.

### 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 템플릿 시스템 | 다양한 교회 템플릿 선택 및 커스터마이징 | P0 |
| 페이지 빌더 | 드래그 앤 드롭 페이지 편집 | P0 |
| 미디어 관리 | 이미지/동영상 업로드 및 갤러리 | P0 |
| 메뉴 관리 | 네비게이션 메뉴 구성 | P0 |
| SEO 최적화 | 메타데이터, 사이트맵 자동 생성 | P1 |
| 다국어 지원 | 다국어 콘텐츠 관리 (i18n) | P1 |
| 블로그/뉴스 | 교회 소식 및 블로그 게시 | P1 |
| 설교 아카이브 | 설교 영상/오디오 아카이빙 | P1 |

### 페이지 구성

```
Homepage
├── 히어로 섹션 (라이브 배너, 환영 메시지)
├── 예배 시간 안내
├── 최신 설교
├── 다가오는 이벤트
├── 소그룹 소개
└── 연락처/위치

About
├── 교회 소개
├── 비전/미션
├── 목회자 소개
├── 교회 역사
└── 시설 안내

Sermons
├── 설교 목록 (시리즈별, 날짜별)
├── 설교 상세 (비디오, 오디오, 노트)
├── 검색 및 필터
└── 시리즈 페이지

Events
├── 이벤트 캘린더
├── 이벤트 상세
├── 등록 폼
└── 과거 이벤트 아카이브

Connect
├── 소그룹 안내
├── 봉사 사역
├── 새가족 환영
└── 연락 폼
```

### 템플릿 시스템

| 템플릿 유형 | 설명 |
|-------------|------|
| Modern | 미니멀, 풀스크린 이미지, 모던 타이포그래피 |
| Traditional | 클래식 레이아웃, 따뜻한 색상 |
| Contemporary | 대형교회 스타일, 비디오 중심 |
| Simple | 심플 레이아웃, 빠른 로딩 |
| Community | 커뮤니티 중심, SNS 통합 |

## 2. Giving Module

### 개요

온라인 헌금 및 기부 시스템으로, 다양한 결제 채널과 정기 헌금을 지원합니다.

### 경쟁력 비교

| 항목 | Tithely | Pushpay | **Digital Church** |
|------|---------|---------|-------------------|
| 카드 수수료 | 3.5% + $0.30 | 2.9% + $0.30 | **2.0% + $0.25** |
| ACH 수수료 | 1.0% + $0.30 | 1.0% | **0.5%** |
| 정기 헌금 | Yes | Yes | **Yes + Smart** |
| 문자 헌금 | Yes | Yes | **Yes** |
| 키오스크 | Basic | Premium | **Included** |
| 주식/암호화폐 | Limited | No | **Yes** |

### 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 온라인 헌금 폼 | 웹사이트 헌금 폼 | P0 |
| 정기 헌금 | 주간/월간 자동 결제 | P0 |
| 다중 펀드 | 일반헌금, 건축헌금 등 | P0 |
| 결제 처리 | Stripe, ACH 연동 | P0 |
| 헌금 영수증 | 자동 발송 (이메일/우편) | P0 |
| 캠페인 | 특별헌금 캠페인 페이지 | P1 |
| 서약 관리 | 헌금 서약 및 추적 | P1 |
| 문자 헌금 | SMS Text-to-Give | P2 |
| 키오스크 모드 | 현장 헌금 키오스크 | P2 |

### 헌금 채널

│  └────┬─────┘  └────┬─────┘  └────┬─────┘  └────┬─────┘   │
│       │             │             │             │          │

│                            │                                │

### 데이터 모델 개요

| 엔티티 | 설명 |
|--------|------|
| Donation | 개별 헌금 트랜잭션 |
| Donor | 헌금자 프로필 (Member 연결) |

| GivingCampaign | 특별 헌금 캠페인 |

### 관리자 대시보드

```
Giving Dashboard
├── 헌금 현황 (오늘, 이번 주, 이번 달, 연간)

├── 펀드별 현황
├── 정기 헌금 현황
├── 최근 헌금 목록
└── 캠페인 진행 상황
```

## 3. Members Module

### 개요

성도 관리 시스템으로, 교인 정보, 가족 관계, 출석, 목회 관리를 통합 제공합니다.

### 경쟁력 비교

| 항목 | Planning Center | Breeze | **Digital Church** |
|------|-----------------|--------|-------------------|
| 성도 프로필 | Good | Basic | **Excellent** |
| 가족 관리 | Good | Basic | **Excellent** |
| 커스텀 필드 | Limited | Basic | **Unlimited** |
| 방문자 후속조치 | Manual | Basic | **Automated** |
| 출석 추적 | Good | Basic | **Multi-method** |
| 성도 디렉토리 | Basic | Good | **Privacy Control** |

### 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 성도 프로필 | 연락처, 사진, 상세 정보 | P0 |
| 가족 관리 | 가족 그룹 및 관계 설정 | P0 |
| 성도 검색 | 고급 검색 및 필터 | P0 |
| 태그/분류 | 성도 분류 태그 시스템 | P0 |
| 방문자 관리 | 새가족 등록 및 후속조치 | P1 |
| 출석 추적 | 예배/이벤트 출석 기록 | P1 |
| 성도 디렉토리 | 온라인 성도 명부 (프라이버시 제어) | P1 |
| 목회 노트 | 심방/상담 기록 (Staff 전용) | P2 |
| 기도 요청 | 기도 요청 접수 및 관리 | P2 |

### 성도 상태 흐름

```
Visitor → Regular Attender → Member → Active
                                  ↓
                              Inactive → Transferred/Removed
```

### 성도 프로필 구성

| 섹션 | 필드 |
|------|------|
| 기본 정보 | 이름, 생년월일, 성별, 연락처 |
| 주소 | 거주지, 우편 주소 |
| 가족 정보 | 가족 구성원, 관계, 역할 |
| 교회 상태 | 회원 상태, 가입일, 세례일, 결혼일 |
| 소그룹 | 소속 소그룹, 봉사 사역 |
| 헌금 | 헌금 이력 요약 (Giving 연동) |
| 출석 | 출석 이력 및 통계 |
| 커스텀 필드 | 교회별 추가 필드 |

### 데이터 모델 개요

| 엔티티 | 설명 |
|--------|------|
| Member | 성도 프로필 |
| Family | 가족 그룹 |
| MemberRelationship | 성도 간 관계 |
| Attendance | 출석 기록 |
| PastoralNote | 목회 노트 (Staff 전용) |
| CareRequest | 심방/기도 요청 |

## 4. Events Module

### 개요

교회 이벤트 생성, 홍보, 등록, 출석 관리를 위한 통합 시스템입니다.

### 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 이벤트 생성 | 이벤트 정보 등록 | P0 |
| 이벤트 캘린더 | 월간/주간 캘린더 뷰 | P0 |
| 온라인 등록 | 이벤트 참석 등록 폼 | P0 |
| 등록 관리 | 참석자 목록 및 관리 | P0 |
| 티켓/유료 이벤트 | 유료 이벤트 결제 | P1 |
| 대기자 명단 | 정원 초과 시 대기자 관리 | P1 |
| 반복 이벤트 | 정기 이벤트 설정 | P1 |
| 이벤트 알림 | 이메일/푸시 리마인더 | P1 |
| 출석 체크인 | 이벤트 당일 체크인 | P2 |

### 이벤트 유형

| 유형 | 설명 |
|------|------|
| Worship Service | 주일/수요 예배 |
| Special Event | 특별 행사, 세미나 |
| Class | 교육 과정, 성경공부 |
| Social | 친교, 야유회 |
| Volunteer | 봉사 활동 |
| External | 외부 행사 |

### 등록 폼 구성

```
Registration Form
├── 개인 정보 (이름, 연락처)
├── 커스텀 질문 (알레르기, 교통 등)
├── 동반자 정보 (가족/어린이)
├── 티켓 선택 (유료 이벤트)
├── 결제 (Giving 연동)
└── 확인 및 이메일 발송
```

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 그룹 생성 | 소그룹/사역팀 생성 | P0 |
| 그룹 검색 | 그룹 디렉토리 및 검색 | P0 |
| 멤버 관리 | 그룹원 추가/제거 | P0 |

## 7. Streaming Module

### 개요

라이브 예배 스트리밍 및 VOD 서비스를 제공합니다.

### 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 라이브 스트리밍 | RTMP/HLS 라이브 방송 | P0 |
| 스트리밍 페이지 | 교회 맞춤 라이브 페이지 | P0 |
| 실시간 채팅 | 시청자 채팅/댓글 | P1 |
| VOD 아카이브 | 녹화본 자동 저장 | P1 |
| 스케줄 방송 | 예약 라이브 설정 | P1 |
| 다중 플랫폼 | YouTube/Facebook 동시 송출 | P2 |
| 시청 분석 | 시청자 수, 시청 시간 분석 | P2 |

### 스트리밍 워크플로우

```
OBS/Encoder → RTMP Ingest → Transcoding → CDN → Viewer
                               ↓
                          DVR/Archive
```

### 연동 플랫폼

| 플랫폼 | 연동 방식 |
|--------|----------|
| YouTube Live | RTMP 동시 송출 |
| Facebook Live | RTMP 동시 송출 |
| Vimeo | 임베드 연동 |
| Custom RTMP | 직접 설정 |

## 8. Check-in Module

### 개요

어린이 체크인 및 보안 시스템으로, 부모/보호자 확인 및 알림을 지원합니다.

### 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 셀프 체크인 | 키오스크 셀프 체크인 | P0 |
| 이름표 인쇄 | 자동 이름표 출력 | P0 |
| 보호자 매칭 | 보안 코드 매칭 시스템 | P0 |
| SMS 알림 | 부모에게 알림 발송 | P0 |
| 가족 체크인 | 가족 일괄 체크인 | P1 |
| 알레르기/의료 정보 | 이름표에 의료 정보 표시 | P1 |
| 출석 리포트 | 어린이 출석 통계 | P1 |
| 모바일 체크인 | 앱 사전 체크인 | P2 |

### 체크인 플로우

```
1. 키오스크에서 전화번호/성도 검색
2. 자녀 선택 및 클래스 확인
3. 보안 코드 생성
4. 이름표 출력 (자녀용 + 보호자용)
5. 자녀 인계 시 보안 코드 확인
6. 체크아웃 완료
```

### 이름표 구성

| 정보 | 위치 |
|------|------|
| 자녀 이름 | 상단 중앙 (큰 글씨) |
| 보안 코드 | 상단 우측 |
| 클래스/방 번호 | 중앙 |
| 알레르기 정보 | 하단 (빨간색 강조) |
| 보호자 전화번호 | 뒷면 |

## 9. AI Shorts Creator Module

### 개요

AI를 활용하여 설교 영상에서 SNS용 숏폼 영상을 자동 생성합니다.

### 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 자동 하이라이트 감지 | AI 핵심 순간 식별 | P0 |
| 클립 생성 | 15-60초 클립 자동 생성 | P0 |
| 자막 생성 | 다국어 자막 자동 생성 | P0 |
| 템플릿 적용 | 브랜딩 템플릿 적용 | P1 |
| 플랫폼 최적화 | TikTok/Reels/Shorts 비율 | P1 |
| 배치 생성 | 다수 클립 일괄 생성 | P1 |
| SNS 직접 게시 | 플랫폼 연동 게시 | P2 |

### AI 처리 파이프라인

```
Input Video → Transcription → AI Analysis → Clip Selection
                                    ↓
              Rendering ← Template ← Subtitle Generation
                 ↓
          Multi-Format Export (9:16, 1:1, 16:9)
```

### 생성 옵션

| 옵션 | 값 |
|------|-----|
| 클립 길이 | 15초, 30초, 60초 |
| 비율 | 9:16 (TikTok), 1:1 (Instagram), 16:9 (YouTube) |
| 자막 스타일 | 하단, 중앙, 애니메이션 |
| 템플릿 | 기본, 미니멀, 생동감, 커스텀 |

## 10. AI Social Studio Module

### 개요

교회 콘텐츠를 기반으로 SNS 게시물을 AI가 자동 생성합니다.

### 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 설교 인용 카드 | 설교 핵심 문구 그래픽 | P0 |
| 이벤트 홍보물 | 이벤트 기반 전단 생성 | P0 |
| 성경 구절 카드 | 오늘의 말씀 그래픽 | P0 |
| 템플릿 라이브러리 | 다양한 디자인 템플릿 | P1 |
| 일정 게시 | 예약 게시 기능 | P1 |
| 캡션 생성 | AI 캡션/해시태그 생성 | P1 |
| 플랫폼 연동 | Instagram/Facebook 직접 게시 | P2 |

### 콘텐츠 유형

| 유형 | 소스 | 출력 |
|------|------|------|
| Quote Card | 설교 텍스트 | 인용 그래픽 |
| Event Flyer | Events 모듈 데이터 | 이벤트 포스터 |
| Bible Verse | 성경 구절 입력 | 말씀 카드 |
| Announcement | 공지사항 입력 | 소셜 그래픽 |
| Story Card | 다양한 콘텐츠 | Instagram Story 형식 |

### 템플릿 스타일

| 스타일 | 특징 |
|--------|------|
| Modern | 미니멀, 산세리프, 그라데이션 |
| Classic | 전통적, 세리프 폰트, 따뜻한 색상 |
| Bold | 강렬한 색상, 큰 타이포그래피 |
| Minimal | 심플, 화이트 스페이스 |
| Custom | 교회 브랜딩 커스텀 |
